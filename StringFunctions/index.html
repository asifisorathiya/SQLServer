What is SET OPERATORS
  • COMBINING ROWS TO GET DATA.
  • SAME NUMBER OF COLUMNS
  • SAME DATA TYPE FOR COLUMNS
  • ORDER BY CLAUSE CAN BE USED BY END OF THE WHOLE QUERY
  • SAME ORDER OF COLUMNS
UNION
  • DISTINCT ROWS AND REMOVES DUPLICATES FROM RESULTSET
  • SELECT A,B FROM TABLE UNION SELECT A,B FROM TABLE;
UNION
  • RETURNS ALL ROWS INCLUDING DUPLICATES
  • SELECT A,B FROM TABLE UNION ALL SELECT A,B FROM TABLE;
EXCEPT
  • RETURNS ROWS FROM FIRST TABLE WHICH ARE NOT AVAILABLE IN TABLE 2
  • SELECT A,B FROM TABLE EXCEPT SELECT A,B FROM TABLE;
EXCEPT
  • RETURNS COMMON ROWS BETWEEN THE TABLES
  • SELECT A,B FROM TABLE INTERSECT SELECT A,B FROM TABLE;
