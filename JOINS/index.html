What is JOIN
  • JOIN IS USED TO COMBINE TWO OR MORE TABLES FOR RECOMBINE DATA, DATA ENRICHMENT AND FILTERING OR DATA ENHANCEMENT.
NO JOIN
  • no join between table
  • SELECT * FROM TABLE; SELECT * FROM TABLE;
INNER JOIN
  • INNER JOIN IS USED TO COMBINE 2 OR MORE TABLE TO RECOMBINE MATCHING DATA
  • SELECT * FROM TABLE_A INNER JOIN TABLE_B ON TABLE_A.ID=TABLE_B.ID;
LEFT JOIN
  • LEFT JOIN IS USED TO COMBINE 2 OR MORE TABLE TO ALL DATA FROM LEFT TABLE AND MATCHING DATA FROM RIGHT TABLE
  • SELECT * FROM TABLE_A LEFT JOIN TABLE_B ON TABLE_A.ID=TABLE_B.ID;
RIGHT JOIN
  • RIGHT JOIN IS USED TO COMBINE 2 OR MORE TABLE TO ALL DATA FROM RIGHT TABLE AND MATCHING DATA FROM LEFT TABLE
  • SELECT * FROM TABLE_A RIGHT JOIN TABLE_B ON TABLE_A.ID=TABLE_B.ID;
FULL JOIN
  • FULL JOIN IS USED TO COMBINE 2 OR MORE TABLE TO ALL DATA FROM BOTH TABLE
  • SELECT * FROM TABLE_A FULL JOIN TABLE_B ON TABLE_A.ID=TABLE_B.ID;
CROSS JOIN
  • CROSS JOIN IS USED TO COMBINE 2 OR MORE TABLE TO ALL DATA FROM BOTH TABLE WITH ALL POSSIBLE VALUES
  • SELECT * FROM TABLE_A CROSS JOIN TABLE_B;
