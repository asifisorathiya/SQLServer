What is Comparison Operator
  • Comparison operator is used to compare data for one value with another one.
=(Equal To)
  • Compare left value to right value in where condition expression
  • SELECT * FROM TABLE WHERE COLUMN=VALUE
!=(NOT Equal To)
  • Compare NOT EQUAL TO left value to right value in where condition expression
  • SELECT * FROM TABLE WHERE COLUMN!=VALUE
>(GREATER THAN )
  • Compare left COLUMN GRAETER THAN right value in where condition expression
  • SELECT * FROM TABLE WHERE COLUMN>VALUE
>=(GREATER THAN EQUAL TO)
  • Compare left COLUMN GRAETER THAN EQUAL TO right value in where condition expression
  • SELECT * FROM TABLE WHERE COLUMN>=VALUE
LESS THAN (LESS THAN )
  • Compare left COLUMN LESS THAN right value in where condition expression
  • SELECT * FROM TABLE WHERE COLUMN < VALUE
LESS THAN= (LESS THAN EQUAL TO )
  • Compare left COLUMN LESS THAN EQUAL TO right value in where condition expression
  • SELECT * FROM TABLE WHERE COLUMN <= VALUE
AND
  • two conditions in where condition expression, ALL CONDITIONS MUST BE TRUE
  • SELECT * FROM TABLE WHERE COLUMN = VALUE and column=value
OR
  • two conditions in where condition expression, ATLEAST ONE MUST BE TRUE
  • SELECT * FROM TABLE WHERE COLUMN = VALUE OR column=value
between
  • Between the range/li>
  • SELECT * FROM TABLE WHERE COLUMN between 1 and 200
IN
  • IN IS USED WHEN MULTIPLE DATA TO BE USED IN WHERE CONDITION
  • SELECT * FROM TABLE WHERE COLUMN IN(1,200)
NOT IN
  • NOT IN IS USED WHEN MULTIPLE DATA EXCLUDES TO BE USED IN WHERE CONDITION
  • SELECT * FROM TABLE WHERE COLUMN NOT IN(1,200)
LIKE
  • LIKE IS USED FOR PATTERN MATCHING IN CONDITION
  • SELECT * FROM TABLE WHERE COLUMN LIKE '%VALUE%
NOT LIKE
  • NOT LIKE IS USED FOR PATTERN MATCHING WHERE NOT IN CONDITION
  • SELECT * FROM TABLE WHERE COLUMN NOT LIKE '%VALUE%
