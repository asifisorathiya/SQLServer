What is INSERT?
  • INSERT command is used to add row data in a table
  • INSERT command can have column name or can leave.
  • INSERT command can used to to add data directly from another table as well.
What is UPDATE?
  • UPDATE command is used to MODIFY THE EXISTNG DATA
  • UPDATE TABLENAME SET COLUMNNAME=VALLUE WHERE CONDITION
  • WHERE CONDITION IS VERY IMPORTANT OTHERWISE IT WILL UPDATE ALL DATA
What is DELETE?
  • DELETE command is used to DELETE TABLE DATA
  • IT WILL DELETE ALL DATA IF NO WHERE CONDITION IS ADDED
  • IT WILL DELETE FEW DATA IF WHERE CONDITION IS ADDED
  • ANOTHER OPTION IS TRUNCATE BUT IT WILL ERASE OF ALL DATA WITHOUT KEEP LOGS
