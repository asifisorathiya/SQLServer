What is CREATE?
  • CREATE command is used to create a table which include column information
  • CREATE TABLE TABLENAME (COLUMNNAME DATATYPE CONSTRAINT)
What is ALTER?
  • ALTER command is used to create a ADD OR DROP COLUMN
  • ADD COLUMN :ALTER TABLE TABLENAME ADD COLUMNNAME DATATYPE CONSTRAINT
  • DROP COLUMN :ALTER TABLE TABLENAME DROP COLUMNNAME
What is DROP?
  • DROP command is used to DROP TABLE
  • DROP TABLE TABLENAME
